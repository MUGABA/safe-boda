## Installation

```bash
  nvm install this installs the node version the application is using.
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test


# test coverage
$ yarn run test:cov
```

## api endpoints

after running the application, Please visit `http://localhost:3000/api#/`
for swagger api descriptions.
